Introduction

Python is a high-level programming language known for readability, simplicity, and versatility.

It is widely used in web development, data science, automation, machine learning, and cybersecurity.

print("Hello World")
  • Easy to learn
  • Readable syntax
  • Large ecosystem
Variables and Types

Variables are containers for storing data values.

Python uses dynamic typing, meaning variable types are determined automatically.

name = "Alice" age = 25 height = 1.75
  • String (str)
  • Integer (int)
  • Float (float)
Control Flow

Control flow determines how code executes based on conditions and loops.

Python provides if statements, for loops, and while loops.

if age >= 18: print("Adult") else: print("Minor")
  • If statements
  • For loops
  • While loops
Functions

Functions allow reusable blocks of code.

They improve maintainability and organization.

def greet(name): return f"Hello {name}"
  • Reusable logic
  • Cleaner code
  • Better structure
Modules and Packages

Modules are Python files containing reusable code.

Packages help organize multiple modules into directories.

import math print(math.sqrt(25))
  • Improved organization
  • Code reuse
  • Scalability
